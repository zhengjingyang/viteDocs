# Cesium Demo

项目基于 vue3 + vite + cesium，已实现常见三维动画场

github地址：https://github.com/tingyuxuan2302/cesium-vue3-vite.git

已有功能

- 粒子效果
  -  下雨
  -  下雪
  -  雾天
  -  火焰
- POI点位
  -  基础打点
  -  聚合
  -  primitive底层打点（性能佳）
  -  primitive聚合（性能佳）
  -  点位动态弹窗
- 第三方服务加载
  -  xyz瓦片
  -  3D Tiles
- 材质
  -  道路闪烁
  -  道路流光效果
  -  辐射圈
  -  圆扩散
  -  四色图
  -  流动的水面
  -  天空盒
- 几何
  -  量测
  -  点线面绘制
  -  态势图、箭头
- 场景
  -  水淹模拟
  -  热力图
  -  时间轴
  -  遮罩反选（边界）
- 分析
  -  天际线分析
  -  高程(限高)分析
